# Project Title:
PythonCarCrudProject

Description: The purpose of this project is to be able to Create, Read, Update, and Delete a Car in inventory
utilizing MapStruct to respond to requests

## Getting Started:

Clone the repository: git clone https://github.com/LeroyOrr/CarCrudProject.git

# Dependencies: Install dependencies
- uv Installation(inside of root): $ curl -LsSf https://astral.sh/uv/install.sh | sh
- mvn install

# Prerequisites
- Maven 3.6+
- JDK 17+
